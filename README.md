> A Vue.js project

## Build Setup

```bash
# install dependencies
yarn install

# serve with hot reload at localhost:8080
yarn serve

# run automated tests in watch mode
yarn test

# run automated tests and generate code coverage report
yarn test:coverage

# build for production with minification
yarn build
```
